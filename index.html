Conectar con TMDB

Para buscar títulos y saber cuándo se estrenan los nuevos episodios, JB's Series usa la base de datos gratuita de TMDB. Necesitas tu propia clave (gratis, 2 minutos):

  1. Entra en themoviedb.org/signup y crea una cuenta gratis.
  2. Ve a tu perfil → Configuración → API.
  3. Pulsa "Solicitar una clave de API", elige uso "Personal" y rellena el formulario corto.
  4. Copia la "Clave de API (v3 auth)" y pégala aquí abajo.
